# Detailed GoDaddy Deployment Guide: TravelTew News

Follow these specific steps to deploy your project.

## 1. Where to find your files
- **Frontend (Website)**: Look in your project folder for a folder named `dist`. This was just generated by me. You will upload the **CONTENTS** of this folder.
- **Backend (API)**: You will upload the `api` folder and the `package.json` file from your project root.

## 2. Uploading the Website (Frontend)
1. Log in to your GoDaddy account and open **cPanel**.
2. Look for the **Files** section and click on **File Manager**.
3. In the left sidebar, double-click **public_html**.
4. Click the **Upload** button at the top.
5. Drag and drop all files from your **LOCAL** `dist` folder into the upload area.
   - *Note: Do not upload the `dist` folder itself, just the files inside it (assets, index.html, .htaccess).*

## 3. Setting up the Backend (Node.js)
1. Go back to the main cPanel dashboard.
2. Search for **Software** section and click **Setup Node.js App**.
3. Click the blue **Create Application** button.
4. **Node.js version**: Select **20.x** or higher.
5. **Application mode**: Select **Production**.
6. **Application root**: Type a name for your backend folder (e.g., `news-backend`). cPanel will create this folder for you in your home directory.
7. **Application URL**: Select your domain and leave the box next to it empty (or type `api` if you want it at `yourdomain.com/api`).
8. **Application startup file**: Type `api/backend/server.js`.
9. Click **Create** at the top right.

## 4. Connecting the Database
Since GoDaddy shared hosting is better with MySQL, we use **Neon.tech** for PostgreSQL:
1. Go to [Neon.tech](https://neon.tech/) and sign up.
2. Create a project named "TravelTew News".
3. Under **Connection Details**, copy the **Connection String** (it starts with `postgres://...`).
4. Go back to your **Node.js App** in cPanel.
5. Under **Configuration variables**, click **Add Variable**.
   - **Name**: `DATABASE_URL`
   - **Value**: (Paste your Neon string here)
6. Click **Save** then click **Restart**.

## 5. Final Step: Install Dependencies
1. In the Node.js App page in cPanel, look for the button **Run JS Install**.
2. Click it and wait for it to finish.
3. Your site is now LIVE!
